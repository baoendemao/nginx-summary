# nginx-summary 
* nginx原理
    * http反向代理服务器
    * 静态资源web服务
    * 负载均衡调度器SLB
    * 动态缓存
    * nginx的优点
        * IO多路复用epoll
        * 代码模块化
* nginx相关文件
    * /etc/nginx  => 配置文件目录
    * /etc/nginx/nginx.conf   => 总的配置文件
    * /var/log/nginx   => nginx log
    * /usr/sbin/nginx   => which nginx
